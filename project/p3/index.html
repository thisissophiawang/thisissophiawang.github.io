Computer Vision-Enhanced Waste Sorting Assistant

This project helps international students and newcomers with waste sorting by providing recycling recommendations and visualizing waste trends through a dashboard, making waste management easier and more accessible. Using computer vision techniques, the system identifies multiple waste categories in real-time and provides data-driven insights to improve recycling habits.

Key features

  • Real-time detection using YOLOv11 to identify recyclable, hazardous, kitchen, and other waste categories in a single image.
  • Bounding box visualizations with confidence scores for each detected waste item.
  • Tailored recycling recommendations based on detected waste types.
  • Interactive data analysis dashboard for waste trend visualization using Gradio.
  • Regional waste data analysis for Burnaby, Coquitlam, Richmond, and Surrey areas.

Performance Metrics

  • 83.49% – mAP@50 (Mean Average Precision)
  • 74.97% – mAP@50-95
  • 78.0% – Recall
  • 81.2% – Precision
