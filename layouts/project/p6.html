NEU Kanbas

Learning Management System for Northeastern University

Project Overview

A custom-built learning management system designed exclusively for Northeastern University, enhancing the online learning experience with dynamic course management and interactive features. This platform leverages modern web technologies to provide an intuitive interface for both faculty and students, facilitating seamless course delivery and academic progress tracking.

REACT REDUX NODE.JS TYPESCRIPT BOOTSTRAP MONGODB

Key Features

Dynamic Course Dashboard

Interactive interface for course overview and navigation, providing instant access to all course materials and activities

Quiz & Assignment Management

Comprehensive system for creating, managing, and grading assignments with real-time feedback capabilities

Module Organization

Structured content management with intuitive drag-and-drop functionality for organizing course materials

Interactive Grade Book

Real-time grade management system with automatic calculations and progress tracking

Role-Based Access Control

Different views and permissions for students and instructors, ensuring secure and appropriate access to course content

Technical Implementation

Frontend Architecture

- React with TypeScript for enhanced reliability
- Redux for centralized state management
- Bootstrap for responsive layouts
- React Router for seamless navigation

Backend Infrastructure

- Node.js/Express.js RESTful API
- MongoDB for data persistence
- JWT-based authentication
- CORS and security validation
