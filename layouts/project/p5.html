FraudMiner

Data Mining for Credit Card Fraud Detection

Project Overview

An advanced data mining system designed to detect fraudulent credit card transactions in real-time, addressing the critical challenge of imbalanced data while providing high accuracy fraud detection. This system leverages sophisticated algorithms and pattern recognition techniques to protect financial institutions and consumers from escalating credit card fraud, which currently causes over $28.65 billion in annual global losses.

Technical Details

Data Challenges Addressed

Severe Class Imbalance

Only 492 fraudulent cases (0.17%) in 284,807 transactions

High Dimensional Feature Space

28 PCA-transformed features with temporal data

Non-Linear Fraud Patterns

Complex relationships requiring advanced modeling

Concept Drift

Evolving fraud techniques requiring adaptive detection methods

Architecture & Implementation

Data Preprocessing Pipeline

  • Feature Transformation: RobustScaler for amount normalization
  • Correlation Analysis: Feature relationship visualization via heatmaps
  • SMOTE Implementation: Advanced sampling for minority class enhancement
  • • Before SMOTE: Fraudulent transactions: 0.17% (492 out of 284,807)
  • • After SMOTE: Transformed to a balanced dataset with a 50-50 distribution, enhancing the model's ability to learn and detect fraud patterns.
  • Stratified Splitting: Maintained class distribution in training and validation sets

Model Implementation & Performance

  • Random Forest Model
  • Achieved:
  • • Accuracy: 99.3%
  • • Confusion Matrix: 63 false positives, 435 false negatives
  • • ROC-AUC Curve: Nearly perfect score of 1.00
  • Random Forest Model with SHAP Analysis
  • Neural Network Model
  • Achieved:
  • • AUC Score: 0.971, demonstrating excellent discrimination ability
  • • Precision: 0.78 for fraud detection
  • • Recall: 0.82 for fraud detection
  • • Mean Probability for Fraud Cases: 0.8184
